# Database Management – Triggers, Indexes and Partitioning

Relational database system for Master Studies Admissions built using Oracle SQL and Java Swing.

## Features

- Normalized and selectively denormalized schema
- Triggers for automatic synchronization
- Stored procedures and packages
- JSON Relational Duality Views
- B-tree and Function-based Indexes
- Horizontal and Vertical Partitioning
- Java Swing GUI for CRUD operations
- JDBC connection to Oracle Database

## Technologies

- Oracle Database 23c
- Oracle SQL Developer
- Java
- Swing
- JDBC
- Apache NetBeans
